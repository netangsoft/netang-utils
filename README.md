## Installation

```bash
npm install @netang/utils --save-dev
```

Install mandatory peer dependencies
```bash
npm install lodash --save-dev
```

Optional Dependencies
```bash
npm install qs --save-dev
npm install resize-observer-polyfill --save-dev
```
